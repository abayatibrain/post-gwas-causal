# ADR-0003: MR as IVW plus a mandatory sensitivity-analysis suite, not a single estimator

Date: 2026-06-17
Status: Accepted

## Context

Mendelian randomization is only as credible as its handling of **pleiotropy** —
an instrument affecting the outcome through a path other than the exposure. A
single point estimate (however efficient) can be badly biased and is never
publishable on its own. The design question is what the MR module should
*return* and *require*.

## Options considered

### Option A — IVW only
- **Pros:** simplest; most efficient under valid instruments.
- **Cons:** a single strong pleiotropic outlier biases it arbitrarily; gives no
  signal that anything is wrong. Scientifically incomplete.

### Option B — One "robust" estimator (e.g. weighted median) only
- **Pros:** tolerant of some invalid instruments.
- **Cons:** less efficient when instruments are valid; still no explicit
  pleiotropy *diagnostic*.

### Option C — IVW as the headline estimate + a triangulating sensitivity suite
- **Pros:** mirrors how MR is actually reported. IVW gives the efficient
  estimate; **MR-Egger** adds a directional-pleiotropy *test* (the intercept);
  **weighted median** gives a breakdown-robust estimate; **Cochran's Q** flags
  heterogeneity. Agreement across all four is the evidence; disagreement is the
  alarm. A `harmonize()` step guarantees allele alignment before any of them
  runs.
- **Cons:** more code and more tests — but each piece is small and independently
  testable.

## Decision

**Option C.** The `mr` package ships IVW (fixed + random), MR-Egger
(slope + intercept test), weighted median (bootstrap SE), Cochran's Q, and a
harmonizer. The CLI `mr` command prints all of them together.

## Consequences

- The design is validated by complementary known-answer tests: IVW equals the
  closed-form weighted slope and recovers the true effect; MR-Egger's intercept
  covers zero under no pleiotropy and is significantly nonzero under injected
  directional pleiotropy; the weighted median is more robust than IVW to a
  single invalid instrument; Cochran's Q detects the heterogeneity that
  pleiotropy induces.
- MR-Egger orients instruments to positive exposure effect (standard coding);
  this is documented because it interacts with how directional pleiotropy must
  be simulated to be detectable.
- Future work: MR-PRESSO outlier removal and multivariable MR (MVMR) slot into
  the same suite.
