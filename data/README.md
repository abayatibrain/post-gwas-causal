# data/

This directory is for **local** GWAS summary statistics, eQTL files, and
reference LD panels. Nothing here is committed (see `.gitignore`) — the toolkit
runs entirely on synthetic data generated by `post_gwas_causal.simulate`, and
real summary-statistics dumps are large and often access-controlled.

Drop harmonized summary statistics (e.g. from the GWAS Catalog or OpenGWAS)
here when running on real loci.
